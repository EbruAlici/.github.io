The difference in '=' vs '==' vs '.equals'

If you wanted to assign the value on the right to the value on the left you would use '='. You would use the '==' if you wanted to compare two values using int, boolean, float or characters. If the value on the left and the right are exactly the same it will return true and if it is not then it would return false.

Examples:

  • 1 == 1 //True
  • 1 == 2 //False
  • "x" == "x" //True
  • "x" == "y" //False
