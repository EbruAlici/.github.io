The difference in '=' vs '==' vs '.equals'

If you wanted to assign the value on the right to the value on the left you would use '='.

You would use the '==' if you wanted to compare two values using int, boolean, float or characters. If the value on the left and the right are exactly the same it will return true and if it is not then it would return false.

Examples:

  • 1 == 1 //True
  • 1 == 2 //False
  • "x" == "x" //True
  • "x" == "y" //False

Math Object

Math is a class built into java that contains several math related static members and methods. Which means you can use them without justifying the class. Names of classes are capitalized in java, so it's typed in as Math and not math.

Math.random() returns a random double value between 0 and 1 including 0 but never 1.

Cast double to an int

A 'Double' holds a value that usually contains a number with a decimal point such as 1.99, 2.99. There are times when you want an integer value. You can change a 'double' to an 'integer' by putting a (int) front of it.

Example:

  • (Int) 11.22 // 11
  • (Int) 12.11 // 12

    • The number after the decimal is dropped and the number before the decimal gets returned

    Example:
    • (Int)(Math.random() * 11) //gives you a range between 0 and 10
    • ((Int)(Math.random() * 20) + 1) // gives you range between 1 and 20
